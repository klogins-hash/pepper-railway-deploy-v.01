# Aria Implementation Guide

Complete step-by-step guide for deploying Aria as your strategic AI co-leader.

## Prerequisites

### Technical Requirements
- **VAPI account** (Pro tier recommended for production)
- **Cartesia voice synthesis** integration
- **Phone number** or web interface for voice interaction
- **API access** for business system integrations (optional)

### Knowledge Requirements
- Understanding of ADHD combined type characteristics
- Familiarity with INFJ personality traits
- Basic voice AI terminology
- Your business context and strategic goals

## Step 1: VAPI Configuration

### 1.1 Create New Assistant

```
1. Log into VAPI dashboard
2. Navigate to Assistants â†’ Create New
3. Name: "Aria - Strategic Co-Leader"
4. Description: "Voice-first strategic business partner for ADHD+INFJ solopreneurs"
```

### 1.2 Configure Voice Settings

**Provider:** Cartesia (recommended)

**Voice Selection:**
- **Tone**: Warm, professional, confident
- **Gender**: Female (based on "Pepper Potts" inspiration)
- **Age**: Mid-30s to 40s professional range
- **Accent**: Neutral American English

**Settings:**
- **Speed**: 1.0x (natural pace - critical for ADHD processing)
- **Stability**: 0.8 (allows natural variation)
- **Clarity**: 0.9 (high clarity for focus)
- **Emotion**: Enable (essential for tone modulation)

### 1.3 Import Prompt

```
1. Copy entire contents of prompts/vapi-prompt.txt
2. Paste into System Prompt field
3. Verify formatting maintained
4. Save configuration
```

### 1.4 Configure Model Settings

**LLM Model:** GPT-4 Turbo (or equivalent)
- **Temperature**: 0.7 (balanced creativity/consistency)
- **Max tokens**: 150-200 (enforces brevity)
- **Top P**: 0.9
- **Frequency penalty**: 0.3 (reduces repetition)
- **Presence penalty**: 0.3

## Step 2: Testing & Validation

### 2.1 Basic Functionality Test

**Test 1: Response Length**
```
User: "What should I focus on today?"
Expected: Response under 20 seconds
Verify: Use stopwatch to time response
```

**Test 2: Strategic Challenge**
```
User: "I'm thinking of offering a huge discount to get more clients."
Expected: Pushback with data/reasoning, alternative suggestions
Verify: Aria challenges decision, doesn't just validate
```

**Test 3: Executive Function Support**
```
User: "I need to launch a new program but I'm overwhelmed."
Expected: Automatic task breakdown into steps
Verify: Clear, actionable next steps provided
```

**Test 4: Conversational Fillers**
```
Listen for: "Well," "actually," "here's the thing," "you know"
Expected: Multiple fillers per response
Verify: Natural speech flow, not robotic
```

**Test 5: Processing Time**
```
User: "Let me think about that..." [pause]
Expected: Aria waits patiently, doesn't interrupt
Verify: No pressure for immediate response
```

### 2.2 Advanced Functionality Test

**Test 6: Tone Modulation**
```
Try: Ask about crisis, then opportunity, then routine task
Expected: Tone shifts appropriately for context
Verify: Can hear emotional expression differences
```

**Test 7: 4-Level Challenge Protocol**
```
Present progressively more problematic decisions
Expected: Escalating challenge intensity
Verify: From curious to firm as stakes increase
```

**Test 8: Values Connection**
```
Ask: "Why should I do this task?"
Expected: Connection to larger vision/purpose
Verify: INFJ meaning-making, not just task completion
```

### 2.3 Integration Testing

**Test 9: Context Retention**
```
Have multi-turn conversation referencing previous points
Expected: Memory of earlier conversation
Verify: No need to repeat information
```

**Test 10: Proactive Initiation**
```
Set up scenario where Aria should surface concern
Expected: Proactive warning/opportunity
Verify: ~40% of interactions Aria-initiated over time
```

**For Consultants:**
```

ROI demonstration, case study development, and client results 
documentation."
```

**For Thought Leaders:**
```
Add to [Identity] section:
"You specialize in thought leadership and content strategy. Your 
focus is on audience building, platform leverage, and message 
amplification."
```

### 3.2 Response Length Tuning

**If user needs shorter responses:**
```
Change in [Response Guidelines]:
From: "DEFAULT: 15-20 seconds maximum"
To: "DEFAULT: 10-15 seconds maximum"

And add: "Aim for single-breath responses as absolute default"
```

**If user can handle longer responses:**
```
Change in [Response Guidelines]:
From: "DEFAULT: 15-20 seconds maximum"
To: "DEFAULT: 20-30 seconds maximum"

Note: Monitor for decreased engagement
```

### 3.3 Challenge Intensity Adjustment

**For users needing gentler pushback:**
```
Modify [Task & Goals] section, Level 1:
Increase emphasis on "What's making you feel..." questions
Reduce directness of challenge statements
```

**For users wanting stronger challenge:**
```
Modify [Task & Goals] section, Level 3:
Use more direct language: "That won't work because..."
Provide data immediately rather than asking questions first
```

## Step 5: Rollout Strategy

### 5.1 Phased Adoption

**Week 1: Learning Phase**
- Use Aria for low-stakes decisions only
- Focus on understanding her personality
- Test different types of queries
- Adjust to voice-first interaction

**Week 2: Strategic Testing**
- Present real business decisions
- Allow Aria to challenge thinking
- Test executive function features
- Gather feedback on response length

**Week 3: Integration**
- Make Aria part of daily routine
- Morning strategic check-ins
- Decision validation
- End-of-day reflection

**Week 4: Full Partnership**
- Rely on Aria for strategic input
- Trust the challenge protocol
- Use proactive features
- Measure business impact
